# Oxy-ify

A fantastic Magisk Module that will turn your device into a OnePlus flagship by adding:
- Oxygen OS boot animation;
- Oxygen OS custom media;
- Oxygen OS apps (Camera, Gallery, Weather);
- OnePlus Slate font.
